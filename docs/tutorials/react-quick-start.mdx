# Tutorial: React/Web Quick Start

**Tutorial** - *Learn the basics by building your first Fishjam web app*

This tutorial will guide you through integrating Fishjam into your React web application step by step.  
By the end, you'll have a working video streaming web app and understand the core concepts.

## What you'll build

A simple React web app that can join video calls and stream video between participants.

## What you'll learn

- How to install and set up Fishjam for React
- How to join a room and start streaming
- How to display video from other participants
- How to manage media devices

## Prerequisites

- React development environment set up
- Access to [Fishjam Dashboard](https://fishjam.io/app)

## Step 1: Install and set up

### Install the package

```bash npm2yarn
npm install @fishjam-cloud/react-client
```

### Set up Fishjam context

Wrap your app in the `FishjamProvider` component:

```tsx
import React from "react";
import ReactDOM from "react-dom/client";
import { FishjamProvider } from "@fishjam-cloud/react-client";

ReactDOM.createRoot(document.getElementById("root")!).render(
  <React.StrictMode>
    <FishjamProvider>
      <App />
    </FishjamProvider>
  </React.StrictMode>,
);
```

### Get your Room Manager URL

1. Log in to [Fishjam Dashboard](https://fishjam.io/app)
2. Navigate to your Sandbox environment
3. Copy your Room Manager URL - you'll need this for testing

## Step 2: Join a room and start streaming

Create a component that joins a room and starts streaming:

```tsx
import { useConnection, useCamera } from "@fishjam-cloud/react-client";

export function JoinRoomButton() {
  const { joinRoom } = useConnection();
  const { start: startCamera } = useCamera();

  const handleJoinRoom = async () => {
    const roomName = "testRoom";
    const peerName = "testUser";

    // Fetch peer token from Room Manager
    const response = await fetch(
      `https://fishjam.io/api/v1/connect/${ROOM_MANAGER_ID}/room-manager/?roomName=${roomName}&peerName=${peerName}`,
    );
    const { url, peerToken } = await response.json();

    // Start camera
    await startCamera();

    // Join the room
    await joinRoom({ url, peerToken });
  };

  return <button onClick={handleJoinRoom}>Join Room</button>;
}
```

## Step 3: Display your video

Show your own video stream:

```tsx
import { useCamera } from "@fishjam-cloud/react-client";

export function MyVideo() {
  const { stream } = useCamera();

  return (
    <video
      ref={(video) => {
        if (video && stream) video.srcObject = stream;
      }}
      autoPlay
      muted
      playsInline
    />
  );
}
```

## Step 4: Display other participants  

Show video from other peers:

```tsx
import { usePeers } from "@fishjam-cloud/react-client";

export function ParticipantsView() {
  const { remotePeers } = usePeers();

  return (
    <div>
      {remotePeers.map((peer) =>
        peer.tracks.map((track) => {
          if (track.type === "Video" && track.stream) {
            return (
              <video
                key={track.id}
                ref={(video) => {
                  if (video) video.srcObject = track.stream;
                }}
                autoPlay
                playsInline
              />
            );
          }
          return null;
        })
      )}
    </div>
  );
}
```

## Step 5: Handle connection status

Monitor your connection:

```tsx
import { useConnection } from "@fishjam-cloud/react-client";

export function ConnectionStatus() {
  const { peerStatus } = useConnection();
  
  return <div>Status: {peerStatus}</div>;
}
```

## Complete example

Here's a complete working app:

```tsx
import { useState } from "react";
import { 
  FishjamProvider,
  useConnection,
  useCamera,
  usePeers 
} from "@fishjam-cloud/react-client";

function VideoCall() {
  const { joinRoom, peerStatus } = useConnection();
  const { start: startCamera, stream: myStream } = useCamera();
  const { remotePeers } = usePeers();
  const [isJoined, setIsJoined] = useState(false);

  const handleJoin = async () => {
    const roomName = "testRoom";
    const peerName = `user_${Date.now()}`;
    
    const response = await fetch(
      `https://fishjam.io/api/v1/connect/${ROOM_MANAGER_ID}/room-manager/?roomName=${roomName}&peerName=${peerName}`,
    );
    const { url, peerToken } = await response.json();

    await startCamera();
    await joinRoom({ url, peerToken });
    setIsJoined(true);
  };

  return (
    <div>
      <h1>Fishjam Video Call</h1>
      <p>Status: {peerStatus}</p>
      
      {!isJoined && (
        <button onClick={handleJoin}>Join Room</button>
      )}

      {myStream && (
        <video
          ref={(video) => {
            if (video) video.srcObject = myStream;
          }}
          autoPlay
          muted
          playsInline
          style={{ width: 300, height: 200 }}
        />
      )}

      <div>
        {remotePeers.map((peer) =>
          peer.tracks.map((track) => {
            if (track.type === "Video" && track.stream) {
              return (
                <video
                  key={track.id}
                  ref={(video) => {
                    if (video) video.srcObject = track.stream;
                  }}
                  autoPlay
                  playsInline
                  style={{ width: 300, height: 200 }}
                />
              );
            }
            return null;
          })
        )}
      </div>
    </div>
  );
}

export default function App() {
  return (
    <FishjamProvider>
      <VideoCall />
    </FishjamProvider>
  );
}
```
