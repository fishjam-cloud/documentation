# Welcome to Fishjam

<p style={{ fontSize: "1.2rem" }}>
Welcome to the Fishjam documentation!

Fishjam is a multimedia streaming toolkit that allows you to build real-time
video and audio streaming applications using WebRTC. We provide the
infrastructure, media server, and client SDKs so you can focus on building
your apps.

</p>
